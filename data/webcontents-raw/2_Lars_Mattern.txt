<h1 id="i-overview">I. Overview<a class="headerlink" href="#i-overview" title="Permanent link">&para;</a></h1>
<p>This week we have to draw a possible final project, so I started playing around with different software.
I&rsquo;ll don&rsquo;t describe &ldquo;pixel&rdquo; paint tools like the windows integrated MS Paint or the open source tool <a href="https://www.gimp.org/">GIMP</a>  </p>
<p>For cutting, engraving and printing are vector graphics and / or 3D design files mandatory.  </p>
<p>Vector graphics are lossless scalable, because it uses &ldquo;nodes&rdquo; and between this nodes are &ldquo;pathes&rdquo;<br />
Commonly uses file formats are <code>svg</code> or <code>dxf</code>  </p>
