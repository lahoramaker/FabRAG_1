<h2 id="arduino-brushless-motor-control-esc">Arduino Brushless Motor Control | ESC |<a class="headerlink" href="#arduino-brushless-motor-control-esc" title="Permanent link">&para;</a></h2>
<h2 id="group-assignment"><strong>Group Assignment</strong><a class="headerlink" href="#group-assignment" title="Permanent link">&para;</a></h2>
<hr />
<p>Measure the power consumption of an output device.</p>
<h2 id="individual-assignment"><strong>Individual Assignment</strong><a class="headerlink" href="#individual-assignment" title="Permanent link">&para;</a></h2>
<hr />
<p>Add an output device to a microcontroller board you&rsquo;ve designed, and program it to do something</p>
<p>Introduction:</p>
<p>For this week I will be making two assignments using an Arduino board. Beacuse of the panemdemic I am using Arduino since I cannot get another board. My two assignmets being first a brushless motor and the second will be a stepper motor.</p>
<h2 id="brushless-motor">Brushless Motor<a class="headerlink" href="#brushless-motor" title="Permanent link">&para;</a></h2>
<p>The components I used are:</p>
<p>1- Arduino Uno</p>
<p>2-RC ESC 3cell</p>
<p>3- Brash Motor KV1700</p>
<p>4- 3S LiPo Battery</p>
<p>5-10k Poten-Tiometer</p>
<p>6-wire</p>
<p>7- Breadboard</p>
<p>I connected all my materials together as shown in the pic .</p>
<p>(ESC Signal)</p>
<p>ESC wire yellow to D9</p>
<p>ESC wire black to GND</p>
<p>ESC wire red to 5V</p>
<p>(ESC Power )</p>
<p>ESC Power In 11.1 V</p>
<p>(10K Poten Tiometer)</p>
<p>Poten Tiometer wire yellow to analog A0</p>
<p>Poten Tiometer wire black to GND</p>
<p>Poten Tiometer   wire red to 5V</p>
<p>(3S Lipo Battery 11.1v )</p>
<p>3S Lipo Battery 11.1v to ESC Motor</p>
<p>(Brushless MOTOR )</p>
<p>Brushless MOTOR TO ESC</p>
<h2 id="operating-the-brushless-motor">Operating the Brushless Motor<a class="headerlink" href="#operating-the-brushless-motor" title="Permanent link">&para;</a></h2>
<p>To start making my Brushless motor I have to start assembling the components that I have with me. All of the components that I used are shown in the list above. I used a 3S Lipo Battery which has 3 cells and that is 11.1 V and that is enough to keep my brushless motor running perfectly. Since the brushless motor is very power hungry we used this 11.1 V battery to give it enough power it needs to work.</p>
<p>The cell battery has 3 wires which forms what is called a Battery Eliminator Circuit that I will explain each one of it&rsquo;s wires and their function. The yellow wire is used for the signal from the board. The red wire is used for the VCC +5V, and the black wire is used for the ground.</p>
<p>The Potentionmeter is used to control the power of the motor by increasing and decreasing the RPM of the brushless motor. I learned about these steps from this <a href="https://howtomechatronics.com/tutorials/arduino/arduino-brushless-motor-control-tutorial-esc-bldc/">link</a>.</p>
<p><img alt="" src="../../images/week12/O1.jpg" /></p>
<p>This picture shows the final assembled circuit I used for this assignment.</p>
<h2 id="code">Code<a class="headerlink" href="#code" title="Permanent link">&para;</a></h2>
<div class="codehilite"><pre><span></span><code>/*

*/
#include &lt;Servo.h&gt;
Servo ESC;     // create servo object to control the ESC
int potValue;  // value from the analog pin
void setup() {
  // Attach the ESC on pin 9
  ESC.attach(9,1000,2000); // (pin, min pulse width, max pulse width in microseconds)
}
void loop() {
  potValue = analogRead(A0);   // reads the value of the potentiometer (value between 0 and 1023)
  potValue = map(potValue, 0, 1023, 0, 180);   // scale it to use it with the servo library (value between 0 and 180)
  ESC.write(potValue);    // Send the signal to the ESC
}
</code></pre></div>

<h2 id="stepper-motor">Stepper Motor<a class="headerlink" href="#stepper-motor" title="Permanent link">&para;</a></h2>
<p>For the Stepper motor I will be listing all the components that I have used to make it below and I learned how to make this project from this <a href="https://www.instructables.com/id/BYJ48-Stepper-Motor/">website</a>:</p>
<p>Arduino Board</p>
<p>Jumper</p>
<p>BYJ48 Stepper Motor 5v</p>
<p><img alt="" src="../../images/week12/O2.jpg" /></p>
<p>Stepper Motor Driver Board ULN2003</p>
<p><img alt="" src="../../images/week12/O3.jpg" /></p>
<p>For the stepper motor I will also be using Arduino board to send signals to control the stepper motor. The stepper motor is used as an output to show the results of the code as the motor rotates clockwise and counter clockwise. Stepper Motor Driver Board is used to give commands from the Arduino board to the stepper motor. This is the drawing sketch of how I aligned my stepper motor and it&rsquo;s components.</p>
<p><img alt="" src="../../images/week12/O4.jpg" /></p>
<p>Now the picture below shows the Stepper board all connected to each other and the video below shows the motor working properly and how I operate it. The connections I made from the Motor Driver Board to the Arduino board are as follows:</p>
<p>I connected the Motor Driver Board Blue wire from in1 to D8 in Arduino Board.</p>
<p>I connected the Motor Driver Board Green wire from in2 to D9 in Arduino Board.</p>
<p>I connected the Motor Driver Board Grey wire from in3 to D10 in Arduino Board.</p>
<p>I connected the Motor Driver Board Yellow wire from in4 to D11 in Arduino Board.</p>
<p><img alt="" src="../../images/week12/O5.jpg" /></p>
<h2 id="code-for-stepper-motor">Code for Stepper Motor<a class="headerlink" href="#code-for-stepper-motor" title="Permanent link">&para;</a></h2>
<div class="codehilite"><pre><span></span><code>/*
   BYJ48 Stepper motor code
   Connect :
   IN1 &gt;&gt; D8
   IN2 &gt;&gt; D9
   IN3 &gt;&gt; D10
   IN4 &gt;&gt; D11
   VCC ... 5V Prefer to use external 5V Source
   Gnd
   written By :Mohannad Rawashdeh
https://www.instructables.com/member/Mohannad+Rawashdeh/
     28/9/2013
  */

#define IN1  8
#define IN2  9
#define IN3  10
#define IN4  11
int Steps = 0;
boolean Direction = true;// gre
unsigned long last_time;
unsigned long currentMillis ;
int steps_left=4095;
long time;
void setup()
{
Serial.begin(115200);
pinMode(IN1, OUTPUT);
pinMode(IN2, OUTPUT);
pinMode(IN3, OUTPUT);
pinMode(IN4, OUTPUT);
// delay(1000);

}
void loop()
{
  while(steps_left&gt;0){
  currentMillis = micros();
  if(currentMillis-last_time&gt;=1000){
  stepper(1);
  time=time+micros()-last_time;
  last_time=micros();
  steps_left--;
  }
  }
   Serial.println(time);
  Serial.println(&quot;Wait...!&quot;);
  delay(2000);
  Direction=!Direction;
  steps_left=4095;
}

void stepper(int xw){
  for (int x=0;x&lt;xw;x++){
switch(Steps){
   case 0:
     digitalWrite(IN1, LOW);
     digitalWrite(IN2, LOW);
     digitalWrite(IN3, LOW);
     digitalWrite(IN4, HIGH);
   break;
   case 1:
     digitalWrite(IN1, LOW);
     digitalWrite(IN2, LOW);
     digitalWrite(IN3, HIGH);
     digitalWrite(IN4, HIGH);
   break;
   case 2:
     digitalWrite(IN1, LOW);
     digitalWrite(IN2, LOW);
     digitalWrite(IN3, HIGH);
     digitalWrite(IN4, LOW);
   break;
   case 3:
     digitalWrite(IN1, LOW);
     digitalWrite(IN2, HIGH);
     digitalWrite(IN3, HIGH);
     digitalWrite(IN4, LOW);
   break;
   case 4:
     digitalWrite(IN1, LOW);
     digitalWrite(IN2, HIGH);
     digitalWrite(IN3, LOW);
     digitalWrite(IN4, LOW);
   break;
   case 5:
     digitalWrite(IN1, HIGH);
     digitalWrite(IN2, HIGH);
     digitalWrite(IN3, LOW);
     digitalWrite(IN4, LOW);
   break;
     case 6:
     digitalWrite(IN1, HIGH);
     digitalWrite(IN2, LOW);
     digitalWrite(IN3, LOW);
     digitalWrite(IN4, LOW);
   break;
   case 7:
     digitalWrite(IN1, HIGH);
     digitalWrite(IN2, LOW);
     digitalWrite(IN3, LOW);
     digitalWrite(IN4, HIGH);
   break;
   default:
     digitalWrite(IN1, LOW);
     digitalWrite(IN2, LOW);
     digitalWrite(IN3, LOW);
     digitalWrite(IN4, LOW);
   break;
}
SetDirection();
}
}
void SetDirection(){
if(Direction==1){ Steps++;}
if(Direction==0){ Steps--; }
if(Steps&gt;7){Steps=0;}
if(Steps&lt;0){Steps=7; }
}
