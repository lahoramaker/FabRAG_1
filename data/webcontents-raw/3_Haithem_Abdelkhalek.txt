        <div class="row">
            <div class="col-sm-12"><br><span style="color:#0B3C5D; font-weight: bold;"><strong>What are the safe / un-safe materials for laser cutting?</strong></span></div>
        </div> 
        
        <div class="row" id="row1">
            <div class="col-sm-12"><br>
                <ul>
                    <li> Before proceeding with laser cutting it was important to identify what are the safe materials to laser cut and what materials are dangerous and should not be used for laser cutting. 
                        <ul>
                        <li><strong>Safe Materials: </strong> Many wood, Plywood/Composite woods, MDF/Engineered woods, Paper, card stock, Cardboard, carton, Cork, Acrylic/Lucite/Plexiglas/PM, Delrin (POM), Kapton tape (Polyimide), Mylar, Solid Styrene, Depron foam, Gator foam, Cloth/felt/hemp/cotton, Leather/Suede, Magnetic Sheet, NON-CHLORINE-containing rubber, Teflon (PTFE), Carbon fiber mats/weave, Coroplast ('corrugated plastic'). </li>
                        <li><strong>Un-Safe Materials:</strong> PVC (Poly Vinyl Chloride)/vinyl/pleather/artificial leather, ABS,HDPE/milk bottle plastic, PolyStyrene Foam, PolyPropylene Foam	Catches fire, Epoxy, Fiberglass, Coated Carbon Fiber, Any foodstuff. </li>
                        </ul>
                    </li>
                    
                    <li>I also found more interesting information about this subject in <a href="http://atxhackerspace.org/wiki/Laser_Cutter_Materials"> This Link</a> and <a href="http://fablabdevon.org/wp-content/uploads/2014/08/Laser_Safe_Materials1.pdf"> This Link</a> .</li>
                </ul>
            </div>
        </div>

            <div class="col-sm-12"><br><span style="color:#0B3C5D; font-weight: bold;"><strong>Vinyl Cutting (Individual Assignment) </strong></span></div>
        </div> 
        
        <div class="row" id="row1">
            <div class="col-sm-12"><br>
                <ul>
                    <li> I decided to cut the FAB ACADEMY logo to add it to the book cover I created earlier using the laser cutter, So the first step was to get the logo as a JPG.</li>
                    <li> In Fab Lab Egypt we are using a Roland GX-24 Vinyl cutter, more info about hte machine can be found <a href="https://www.rolanddga.com/products/vinyl-cutters/camm-1-gs-24-desktop-vinyl-cutter">HERE</a></li>
                    <li>I went through the steps provided by <a href="http://wiki.fablab.is/wiki/How_to_use_the_Roland_GX-24">THIS TUTORIAL</a> and it was really helpful.</li>
                    <li>Turn the machine on and feed the material through the slot. When you are feeding the material, make sure the lever is pushed away from the machine (so it's open).</li>
                </ul>
                <ul style="list-style-type: none;">
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_vinyl_cutter_setup_01.jpg" alt=""></li>
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_vinyl_cutter_setup_02.jpg" alt=""></li>
                </ul>
                <ul>
                    <li> The roll should be placed in the center of the machine. Position the 2 little wheels on both edges of the material. You can slide the wheels easily by pushing the metal plates on the back of the machine.</li>
                    
                    <li> Align the material with the wheels, then pull the lever toward you.</li>
                </ul>
                <ul style="list-style-type: none;">
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_vinyl_cutter_setup_03.jpg" alt=""></li>
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_vinyl_cutter_setup_04.jpg" alt=""></li>
                </ul>
                 <ul>
                    <li>  In the menu display. Use the arrows to select [edge]. Which option you choose from the menu depends on what material you are loading.In my case I chose Piece.</li>
                     <ul>
                          <li><strong>Roll </strong> is for setting up the starting point manually.</li>
                          <li><strong>Piece</strong> is for letting the machine measure all sides of your loaded sheet (like with smaller 'leftover' pieces)</li>
                          <li><strong>Edge</strong> is for letting the machine only find only 1 side of your loaded material. (when you just want to find the starting point at the edge of your material.)</li>
                     </ul>
                     
                     <li>To confirm, push the [enter] button. The machine will now start moving. The machine will pull on the material to find the edge. Now we are ready to push our design to be cut!</li>
                </ul>
                <ul style="list-style-type: none;">
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_vinyl_cutter_setup_05.jpg" alt=""></li>
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_vinyl_cutter_setup_06.jpg" alt=""></li>
                </ul>
                <ul>
                    <li>The Roland vinyl cutter uses as software named "Roland Cut Studio", I tried to import DXF but surprisingly I discovered that it does not accept DXF and instead it accept normal JPG file.</li>
                </ul>
                 <ul style="list-style-type: none;">
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_cutstudio_01.JPG" alt=""></li>
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_cutstudio_02.JPG" alt=""></li>
                     <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_cutstudio_03.JPG" alt=""></li>
                </ul>
                <ul>
                    <li>I imported the FABLAB Logo JPG file to the roland cut studio, right click on the image and selected "Image Outline", then in the resulting window I chose Extract contour lines. </li>
                </ul>
                <ul style="list-style-type: none;">
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_cutstudio_05.JPG" alt=""></li>
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_cutstudio_07.JPG" alt=""></li>
                </ul>
                <ul>
                    <li>click on the print icon, cutting setup then choose the Roland from the printers menu. click on properties and in the resulting window let the machine identify the size of the sheet by clicking "Get from the sheet".</li>
                </ul>
                <ul style="list-style-type: none;">
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_cutstudio_08.JPG" alt=""></li>
                    <li><img style="padding:1px; border:1px solid #1D2731;" class="img-fluid rounded mb-4" src="images/w4_cutstudio_09.JPG" alt=""></li>
                </ul>
                <ul>
                    <li>Now we are ready to cut!.</li>
                </ul>
