<h1 id="17-wildcard-week">17. Wildcard week<a class="headerlink" href="#17-wildcard-week" title="Permanent link">&para;</a></h1>
<p>This week was a lot of fun and allowed us to show off a personal side similar to CNC Week! I was really excited to be able to use the embroidery side of my machine. One of the things I have always been concerned with about embroidery is the cost to have the files created or purchased. And now, unfortunately, many of the sellers on sites such as Etsy are selling trademarked items without the approval to do so!</p>
<p>Being able to learn how to create my own files and how to digitize them correctly will make owning this machine even more productive. I do wish we could get the 8-head machine in the lab working flawlessly, but anyone who knows anything about embroidery machines knows how finicky they are. The wrong tension or setting and you&rsquo;re left with a tangled mess.</p>
<p>My machine only stitches one color of thread at a time, but the concept is still the same. You still have to take an image and digitize it to be able to use it with the machine. I&rsquo;ve laid out the steps to create a new file first; followed by how to use the machine to create the image on a piece of sample material (stabilizer).</p>
<p>I used the InkStitch extension for InkScape. Before beginning the software, I quickly did some research to make sure I would be able to use it to create the correct file extension for my machine. After a little bit of digging, I learned I needed to save my files as PES files, and InkStitch allows me to do so.</p>
<p>I used this <a href="https://inkstitch.org/docs/install-macos/">link</a> to install the extension on my computer. It was a very simple process. Once installed, I deleted the installer file and was ready to begin.</p>
<p><img alt="" src="../../images/week18/WILDSuccessfulInstallation.jpg" title="Inkscape installation complete" /></p>
<p>To learn how to use the sofware, I followed the tutorial for <a href="https://inkstitch.org/docs/basic-usage/">basic usage</a>. To tie in my work from earlier in the class, I decided I wanted to try to embroider my dress design from <a href="/2022/labs/charlotte/students/nidhie-dhiman/assignments/week04">week 4</a> that I used to make my vinyl sticker. I began by importing the picture of the dress I saved from a Google search into Inkscape. Once imported, I knew I needed to convert it to a vector, so I bitmap traced it.</p>
<p><img alt="" src="../../images/week18/WILDTraceBitmap.jpg" title="Tracing the image as a bitmap" /></p>
<p>Next, I selected all the parts of the dress. From the main menu, I clicked on Extensions &ndash;&gt; Ink/Stitch &ndash;&gt; Params. This design was a simple one, so the simulator did it&rsquo;s magic and created the stitches. You can change the values and the simulator will update it as you do so. To understand what each of the values mean, I referred to this chart from the InkStitch Github page.</p>
<p><img alt="" src="../../images/week18/RC-WILDParameterSettings1.jpg" title="Settings pulled from GitHub to help decipher parameters for the machine" /></p>
<p>However, the simulator did such a great job of creating everything I didn&rsquo;t need to really change anything at all. You can view the creation of the file with params below. Once you are satisfied with the way your file is in the viewer, close the window without saving it. You will actually save it in the next step!</p>
<video src="../../images/week18/WILD DressParams.mp4 " controls="controls" style="max-width: 480;"></video>

<p>When you are back in the main InkScape screen, go to the main menu and select File &ndash;&gt; Save As&hellip; From here, name your file and save it with the extension you need for your particular machine. In my case, Brother machines use a .PES file extension. I saved my file this way and then I also saved it as a plain .SVG file in case I needed to come back and make any other changes to it later on.</p>
<p>My file was now ready to go. I saved it on a little USB stick and plugged it into my machine.</p>
<p>I&rsquo;m using a Brother SE600 which is a combination sewing and embroidery machine in one. You change out the base attachment to embroider and this is what allows it to move the embroidery hoop along the X and Y axis.</p>
<p>Since I&rsquo;ve used my machine plenty of times to sew, I quickly loaded up my bobbins with the correct color of matching thred (for embroidery, your bobbin color and top thread color must match), inserted it into the bobbin &ldquo;port&rdquo; and loaded the thread. Next, I needed to learn how to use the embroidery attachment on the machine.</p>
<p>I began by watching <a href="https://www.youtube.com/watch?v=N0yPILKhHVc&amp;t=410s">this video</a> to learn how to use the embroidery function. The tutorial was easy to follow and clear with instructions. Once I had watched it a few times, I moved on to loading my design. Before doing so, I decided to do a test with one of the built in designs that came pre-loaded with the machine. And I am so happy I did! Somewhere I didn&rsquo;t have the thread loaded correctly or the bobbin inserted the wrong way because when my needle started, it resulted it in knotted mess underneath the main needle housing and actually set off the emergency motor lock!</p>
<p><img alt="" src="../../images/week18/WILD-tangled%20mess.jpg" /></p>
<p>At this point I had no option but do remove the needle housing and the two main screws that hold everything in place and start pulling out the tangled thread. I cleaned it all out and worked on setting everything back up slowly. I got myself back to the point where I could try again. I reloaded the bobbin, reloaded the thread - this time taking my time and adjusting everything carefully.</p>
<p>This is when I also realized I had adjusted the tension the last time I used the machine because I was using a much thicker fabric, so I changed that back to &lsquo;4&rsquo;. </p>
<p>I loaded the stabilizer fabric into the embroidery hoop and locked it into place, making sure to lay the tail of the thread on top of the fabric. Next, I clicked through the options on the touch screen to load my file. I didn&rsquo;t need to make any changes to it, so it was a simple process.</p>
<p>Once I was ready, I lowered the presser foot. The indicator light on the start button changed that it was ready to go. I pressed the button and watched the magic happen!</p>
<video src="../../images/week18/WILD completed embroid.mp4" controls="controls" style="max-width: 480;"></video>

<p>Here is the final product!</p>
<p><img alt="" src="../../images/week18/RC-WILDFinalEmbPic.jpg" /></p>
