<h2 id="1-evaluate-and-select-2d-and-3d-software">#1. Evaluate and select 2D and 3D software</h2>
<h3 id="adobe-illustrator">Adobe Illustrator</h3>
<p>To make it a little bit easier on myself I am first going to make a sketch in a program I know really well. This will allow me to be more creative and less frustrated. This basic model I can then use to recreate in other programs.</p>
<h4 id="installing">Installing</h4>
<p>This was easy using the Adobe installer. And I already had the program... so...</p>
<p>I made a drawing of the model as I think it should be. As colours are important, and I drew inspiration from the '80 synth TR-808, I pasted a picture of that synth in my file. 
I then traced the bitmap image so I could easily extract colour from it.</p>
<p><img alt="" src="../../files/week02/colors_and_swatches.jpg" /></p>
<p>I played with different options to make this clock.
The three options I consider can be seen in the illustrator file.</p>
<p><img alt="" src="../../files/week02/lightclock_sideview_ai.png" /></p>
<p>You can download the <a href="../../files/lightclock.ai">original file here</a></p>
<h3 id="photoshop">Photoshop</h3>
<h4 id="installing_1">Installing</h4>
<p>Photoshop was already on my machine so that was easy.</p>
<h4 id="mockup">Mockup</h4>
<p>I made a picture of where I want my machine to be in the house (next to my front door) and added the file I made in Fusion360 to it to get an idea of how it might look in real life</p>
<p><img alt="" src="../../files/week02/bitmap_render.jpg" /></p>
<p>The original design file is over 8MB and I am not going to waste my precious repo space on that. After all, it is a picture with a render placed in it. These two steps are not worth to document other then by words. Which I have now done.</p>
<h3 id="fusion360">Fusion360</h3>
<p>Research into F360 was done with the help from:</p>
<ul>

<p>I think I am now ready to plunge in. I have a slightly new idea for my final project, so I will use Fusion360 to show you what it is.  But to give you a head start</p>
<ul>
<li>The model is more horizontal then vertical now (so it can fit above my front door)</li>
<li>The model should get a nice 80's electronics feeling to it. Think [Roland TR-808] color scheme.

<li>I want to achieve color by filters, not color by LED's. All LED's should be white.</li>

<h4 id="installing_2">Installing</h4>
<p>Installing was a breeze thnx to the license we got with our education. I am now gonna use this to make my first drawing.</p>

The main problem here was getting selecting the metal (which was resolved by hiding the other body in front of it)
The other problem was lifting it, but this was resolved with the move command.</p>

<ul>
<li>copy-paste is not so easy: if I select something, and use cmd-c and cmd-v nothing happens.</li>
<li>I need to change the functions of my pen to be able to use it with Fusion360. But changing the pen is really annoying because I am so used to it to work in a certain way.</li>
<li>I am not used to thinking in 3D on a computer... My designs are always 2D. I get lost in space. </li>
<li>Feeling frustrated and tired. Gonna take a break.</li>

<h4 id="parameters-are-cool">Parameters are cool</h4>
<p><em>Parameters are for 3D objects, what paragraph styles are for text</em>
Playing with the parameters brought me a lot. It is a nice, reassuring feeling I can modify stuff in one place and have it changed everywhere.
I even experimented a little with parameters in parameters.</p>
<p><img alt="" src="../../files/week02/meta-parameter.jpg" />
<img alt="" src="../../files/week02/yodawg.jpg" /></p>
<h4 id="adding-a-little-hole">Adding a little hole.</h4>
<p>I was looking and think how to make a little hole in the back of the acrylic pane. But when I right-clicked, Fusion offered me the 'hole feature.. </p>
<p><img alt="" src="../../files/week02/holefeature.jpg" /></p>
<p>Using this feature I made a tiny little hole in the back, that I will hope will fit a LED. But if not, no worries; I used all parametric numbers, so I can easily change them when needed. Woohoo!</p>
<p><img alt="" src="../../files/week02/parametrichole.jpg" /></p>
<h4 id="multiplying-a-part">Multiplying a part.</h4>
<p>Now I want to get multiple parts of the little thingy I made. Every part should be 5mm longer and on top of the previous one. </p>
<p>Using copy paste on the body I wanted to duplicate, I was able to create a new body. I could also move that on top of the existing body. But now come the resizing. I am using the scaling option from the 'Modify' tools. But this does not feel correct. It will destroy the parametric qualities of my design... </p>
<p><img alt="" src="../../files/week02/scalingfunction.jpg" /></p>
<p>I think I'll have to duplicate my sketch, not the  body. And create a new body from that new sketch... Let's try that.</p>
<p><img alt="" src="../../files/week02/copyingthesketch.jpg" /></p>
<p>Copying the sketch did work and I can now edit the sketch. The problem is that certain properties need to be applied again: the colour and the hole in the back are not copied over. </p>
<p>I should try to find a 'best of both worlds'.</p>
<p>After some testing it seems that copying bodies is the better way to go.</p>

<h4 id="copypaste-bodies-and-destroy-aspiration">Copy/paste bodies and destroy aspiration</h4>
<p>Although I think Fusion360 is the most user friendly 3D design tool out there, it still poses me with a lot of challenges.
But I am currently left with this:</p>
<p><img alt="" src="../../files/week02/state4.jpg" /></p>
<p>And the most important things I learned are:
- set parameters as you go; new stuff comes up when you draw, and whenever you can use the parameters to define the dimensions
- you can make a hole and the parameters for its dimensions
- you can use the move tool to make copies of bodies.
- in my case it seems best to do the <em>chamfers</em> Ã¡fter I made the initial design. On the other hand I am no longer sure if I should do them at all; maybe a more cubic design is better to mimic an '80's design. 
- I am still unsure how to use the sketches. Shall I make one for every body, or can I use one sketch for all similar bodies. 
- How come I have now multiple bodies on my drawing and only three sketches... ?</p>

<h4 id="newer-beginnings">Newer Beginnings</h4>
<p>In order to make fewer mistakes, I am now going to follow a path that Michelle layed out for me. </p>
<p>Steps in this order;</p>
<ol>
<li>Create parameters. </li>
<li>Make sure parameters reference each other where possible.</li>
</ol>
<p><img alt="" src="../../files/week02/parameters2.jpg" /></p>
<ol>
<li>Stay parametric in your design! Also when you think you can't, you can.</li>
<li>Use the same sketch to make multiple bodies.</li>
<li>Use the <em>Pattern on Path</em> option to create multiple copies of a body.</li>
</ol>
<p>All went fine so far.</p>
<p>The main problem now is that I have linked bodies (because of step 5) and when I change color of one, they all change. I need to unlink this somehow... </p>
<p>For now this is the end of the design-process. I hope I can continue and apply what I have learned so far to designs that are meant to be cut or printed by real machines soon.</p>

<p>The FreeCad developers package the current macOS pre-releases with Conda. It is easy to install MiniConda on your Mac and use it to install FreeCad.  </p>

<h4 id="miniconda-interlude">MiniConda interlude</h4>

Which has a tiny little brother for Mac, called miniconda. Let's get and install that.</p>
<p><img alt="" src="../../files/week02/miniconda.jpg" /></p>
<p>After that is that we are going to do some terminal work. </p>

$ conda create --name fcenv --channel conda-forge freecad
