        <h2 class="w3-text-grey w3-padding-16"><i class="fa fa-certificate fa-fw w3-margin-right w3-xxlarge w3-text-teal"></i>Wildcard</h2>
          <p>Week Assessment :	 
          <p style="text-indent: 9em;"> Design and produce something with a digital fabrication process</p>
          
          <p>For this week assignment, I learned how to use Janome Memory Craft 15000. So I started this week with a session from <a href="https://wendwarge.github.io/" target="_blank">Wendy</a>  on how to use the machine.</p>
              
              <img src="media/wildcard/session.jpg" >
              
          <p>Then Mr.Tarek and  <a href="http://fabacademy.org/2018/labs/fablabuae/students/zahrah-alyahyaee/" target="_blank">Zahrah</a> showed me how to prepare the file for the embroidery machine in Digitizer.</p>
    
     <img src="media/wildcard/Digitizer.png" >
    <p>I decided to use the file that I created for the computer-controlled cutting week. So I created new and I chose hoop SQ14. This is all the hoop sizes I got this image from the<a href="https://www.janome.com/siteassets/support/manuals/embroidery-models/inst-mc15000-enlr2.pdf" target="_blank">   manual</a> of the machine.   </p>
        
           <img src="media/wildcard/hoopssize.png" >
        
        <p>Toolboxes > Auto-Digitizer > Insert Artwork.</p>
      <img src="media/wildcard/insertfile.png" >
    
   <p> I selected my image > Click-to-fill> Then I clicked at the place that I want and I select the stitch type and I chose Cross stitch. My design is really simple I don't have to do anything else. </p>
                <img src="media/wildcard/selectstitch.png" >
          
          <p>  Settings up machine was not easy we as students we don't realize how hard is that. Connecting the machine to PC throw wire to send an order to the machine not always work. That why I saved my file in flash memory and I insert the flash in the machine in port 1. 
</p>
          
                <img src="media/wildcard/insertfileinmachine.jpg" >
                  
          <p>On page 104 in the <a href="https://www.janome.com/siteassets/support/manuals/embroidery-models/inst-mc15000-enlr2.pdf" target="_blank">   manual</a> shows how to enable the embroidery mode and it shows the steps of how to place the embroidery unit on the back side of the machine base. </p>
          
          <p>Before that <a href="https://wendwarge.github.io/" target="_blank">Wendy</a>  showed me to set up the fabric for the machine so what I did I had to put paper under the fabric  and what she taught me was exactly what I found in machine manual.   </p>
          
          <img src="media/wildcard/settingfabric.png" >
          <p>And this what I did. </p>
               <img src="media/wildcard/settingfabricwithmachine.jpg" >
          
          <p>I selected my design then I pressed the start button on the machine </p>
