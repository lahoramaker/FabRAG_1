                <p>My task for this week was to build laser cutting images with Inkscape, using Ari's 3D model as basis. I imported the model, and used measuring tool for dimensions.</p>
                <div class="imgbox">
                    <img class="answerimgs" src="Images/week_16/fusion_1.jpg">
                    <div class="imgoverlay">
                        <div class="imgtxt" >The Machine - 3D</div>
                    </div>
                </div>
                <br> 
               
               
                <p>On Inkscape, I installed <a href="https://github.com/paulh-rnd/TabbedBoxMaker">Tabbed Box Maker</a> extension, by simply copying the files to inkscape/shares/extensions folder.</p>
                
                <p>Under extension, laser tools, selected tabbed box maker.From opening windows I could select dimensions, tab width, kerf, joint clearance etc. </p>
                <div class="imgbox">
                    <img class="answerimgs" src="Images/week_16/tabbed_box_maker.jpg">
                    <div class="imgoverlay">
                        <div class="imgtxt" >Tabbed options</div>
                    </div>
                </div>
                <br> 
                <p>The extension could only help me out so far, and had to manually split boxes to make the design as I wanted. Measuring the 3d model and then applying values to inkscape. After everything was done, I placed them so we could cut this in 1 go.</p>
