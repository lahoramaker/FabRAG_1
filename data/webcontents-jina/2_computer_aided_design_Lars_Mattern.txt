This week we have to draw a possible final project, so I started playing around with different software. I’ll don’t describe “pixel” paint tools like the windows integrated MS Paint or the open source tool [GIMP](https://www.gimp.org/)

For cutting, engraving and printing are vector graphics and / or 3D design files mandatory.

Vector graphics are lossless scalable, because it uses “nodes” and between this nodes are “pathes”  
Commonly uses file formats are `svg` or `dxf`
